已知限制

  1. Image upload OCR：UI 中“图片输入”入口存在，但后端 OCR 尚未实现，点击后提示“coming soon”。
  2. calculus-result mobile-320 溢出检测 false positive：在 320 px 视口下 scrollWidth > clientWidth 差值约为 47 px，且未定位到任何 offender。结合 html { overflow-x: hidden } 与滚动条占位分析，判定为浏览器度量误报，不影响实际布局。
  3. 本地验证降级：当 DEEPSEEK_API_KEY 未配置时，验证逻辑会降级为本地符号计算，部分复杂表达式会返回 “Cannot verify locally”。
